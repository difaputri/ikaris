# 🛡️ IKARIS

> Cyber Academy Module 3 Project

## 📖 Overview

IKARIS is a web-based project developed as part of the Cyber Academy Module 3 assignment. The application is built using Node.js and Express.js, focusing on implementing backend web development concepts and basic web application deployment.

The project demonstrates fundamental web application architecture, server-side programming, and static asset management.

---

## ✨ Features

- Web-based application
- Built with Express.js
- Static asset management
- Responsive interface
- Simple backend routing
- Docker support for deployment

---

## 🛠️ Tech Stack

- Node.js

- HTML
- CSS

- Docker

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/difaputri/ikaris.git
```

Go to the project directory

```bash
cd ikaris
```

Install dependencies

```bash
npm install
```

Run the application

```bash
npm start
```

The application will be available at

```
http://localhost:3000
```

---

## 🐳 Run with Docker

Build Docker image

```bash
docker build -t ikaris .
```

Run container

```bash
docker run -p 3000:3000 ikaris
```
